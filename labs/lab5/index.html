SPVCE

Responsive background image: The background uses a repeating star image that starts from the top left and fills the screen.

Srcset: Resolution Swapping

The hero image uses two different resolution photos. The resolution changes depending on the size of the browser window.

Object-fit

The image uses object-fit: cover and object-position: 95% 50% to keep the astronaut in focus. The astronaut remains in frame no matter the size of the browser.

Art Direction Swapping

In a mobile layout, a smaller vertical photo is displayed for the viewer. When the website is viewed on desktop, a larger horizontal photo of the same subject is displayed instead.

SVG: The website "logo" and footer background use SVGs to make the website more interesting without compromising image size or resolution.
